# Kubernetes Scheduling and Pod Management Lab

This repository contains hands-on Kubernetes tasks covering core scheduling and pod management concepts.

## Topics Covered
- nodeName scheduling
- nodeSelector
- Taints and Tolerations
- Node Affinity (required & preferred)
- QoS Classes (Guaranteed, Burstable, BestEffort)
- DaemonSet
- Static Pods

## Tasks
- Task 1: nodeName
- Task 2: nodeSelector
- Task 3: Taints & Tolerations
- Task 4: Node Affinity
- Task 5: QoS Classes
- Task 6: DaemonSet
- Task 7: Static Pod
